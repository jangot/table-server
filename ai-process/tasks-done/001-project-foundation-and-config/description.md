# Задача: Основа проекта и конфигурация

## Суть задачи

Инициализировать Node-проект на TypeScript с модульной структурой, ввести конфигурацию только из переменных окружения с валидацией и типизацией, настроить логирование, поднять idle-страницу на Express с EJS, добавить проверку наличия Chrome/OBS при старте и заложить оркестрацию последовательного запуска внешних приложений (без реализации самих модулей Chrome/OBS).

## Детали и требования

- **Структура:** `src/`, `src/modules/` по правилам проекта; точка входа; сборка или запуск через ts-node.
- **Конфигурация:** только из env (путь к Chrome, путь к OBS, порт и путь к шаблонам idle-страницы, порт DevTools, таймауты Chrome/OBS, токен бота, список разрешённых пользователей, политика рестартов, уровень логов и т.д.). Отдельный модуль конфигурации: чтение env, валидация (наличие обязательных переменных, допустимые форматы), типизированный объект конфига (интерфейс TypeScript). При ошибке валидации — понятное сообщение и выход.
- **Логирование:** уровни (info, warn, error, debug), ключевые события (старт сервиса, запуск Chrome/OBS, команды, ошибки).
- **Idle-страница:** HTTP-сервер на Express; страница ожидания через EJS (путь к шаблонам и порт из конфигурации). URL этой страницы — стартовый для Chrome и целевой по команде `/idle`.
- **Проверка ПО при старте:** проверка наличия исполняемых Chrome и OBS по путям из конфигурации; при отсутствии — понятное логирование и выход или отказ от запуска соответствующих подсистем.
- **Оркестрация:** механизм последовательного запуска (оркестратор/главный цикл): запуск первого приложения → ожидание сигнала «ready» от модуля → запуск следующего. Порядок и список приложений расширяемы (в MVP: Chrome, затем OBS). Детальная реализация модулей Chrome/OBS — на следующих этапах.

## Контекст

- Документ плана: `docs/plan-execution.md`, Этап 1.
- Целевая ОС: Linux. Конфигурация только из env, без файлов конфига.
- Результат этапа: приложение стартует, валидирует env, поднимает idle-сервер (Express/EJS), пишет логи, при отсутствии Chrome/OBS сообщает понятные ошибки. Без реального запуска Chrome/OBS до следующих этапов.
