# Задача: Команда /help в Telegram боте

## Суть задачи
В Telegram боте необходимо добавить специальную команду `/help` (или аналогичную), которая возвращает пользователю список доступных команд с их описанием. Это позволит пользователю узнать, какие команды поддерживает бот, не обращаясь к документации.

## Детали и требования
- Добавить команду `/help` в Telegram бота
- Команда должна возвращать список всех доступных команд с кратким описанием каждой
- Описания команд должны быть понятны конечному пользователю

## Контекст
- Telegram бот реализован в рамках задач 006 (MVP) и 007 (расширенные команды)
- Существующие команды бота находятся в соответствующем модуле проекта

## Открытые вопросы
- Формат вывода: простой текст или Markdown?
- Нужно ли группировать команды по категориям?
- Регистрировать ли команды в BotFather (через `setMyCommands`)?
