# Задача: Telegram-команды для управления сценами OBS

## Суть задачи

Добавить в Telegram-бота команды для ручного управления сценами OBS: список сцен, переключение по имени, текущая сцена, быстрые команды backup/default и расширить /status отображением текущей сцены. Все команды доступны только whitelisted пользователям (существующий механизм ALLOWED_TELEGRAM_USERS).

## Детали и требования

- **Команды (§7 требований):**
  - **`/scenes`** — показать список сцен (имена и при наличии конфига — подписи).
  - **`/scene <name>`** — переключить сцену OBS на указанное имя. Пример: `/scene chrome`. При несуществующей сцене — сообщение об ошибке.
  - **`/current`** — показать текущую сцену OBS.
  - **`/backup`** — переключить на сцену `backup`.
  - **`/default`** — переключить на сцену `default`.
  - **`/status`** — расширить текущий ответ: добавить статус OBS (connected/disconnected) и текущую сцену. Пример: `OBS: connected`, `Current scene: chrome`.
- **Авторизация:** только пользователи из `ALLOWED_TELEGRAM_USERS` могут вызывать эти команды. Неавторизованные — отказ и логирование попытки (§11, §13).
- **Ошибки:** при недоступности OBS или несуществующей сцене — отправлять пользователю понятное сообщение в чат.
- **Логирование:** команды сцен и отказы неавторизованным логировать в формате key=value (как в задаче 012).
