# Задача: Скрипты по доменам/URL при открытии ссылок в Chrome

## Суть задачи
На некоторых сайтах для попадания на нужный экран необходимо выполнить дополнительные действия (логин, клики, принятие соглашений и т.п.). Нужно добавить возможность привязывать скрипты к доменам или конкретным URL, которые выполняются в Chrome при открытии соответствующей страницы.

## Детали и требования
- Поддержка скриптов, привязанных к доменам или адресам.
- Скрипты выполняются при открытии ссылки в Chrome (или при загрузке страницы).
- Цель — автоматизировать действия для достижения нужного экрана на сайте.

## Контекст
- Управление Chrome в проекте: 002-chrome-launch-and-control, 003-url-switch-via-cdp. CDP (Chrome DevTools Protocol) подходит для внедрения скриптов и выполнения действий на странице.

## Открытые вопросы
- Формат хранения скриптов (файлы по домену, конфиг с маппингом URL → скрипт)?
- Язык скриптов: JavaScript в контексте страницы, последовательность CDP-команд или что-то иное?
- Нужна ли задержка перед выполнением скрипта (ожидание загрузки страницы)?
