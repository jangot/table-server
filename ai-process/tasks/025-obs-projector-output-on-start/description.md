# Задача: Трансляция OBS на проектор (второй экран) при запуске

## Суть задачи
При ручной настройке OBS трансляция на второй монитор (проектор) работает, но после перезапуска приложения трансляция не восстанавливается. Нужно обеспечить автоматическое включение вывода на второй экран при старте OBS, предпочтительно через WebSocket API.

## Детали и требования
- Трансляция на второй монитор настраивается вручную в OBS и при этом работает.
- После перезапуска OBS трансляция не осуществляется.
- Требуется автоматическая настройка/включение трансляции при запуске (например, через OBS WebSocket).

## Контекст
- OBS, второй экран/проектор как выход.
- В проекте есть интеграция с OBS: 004-obs-launch-and-control, 012-obs-websocket-and-scene-service.

## Открытые вопросы
- Какой именно тип вывода в OBS используется (полноэкранный проектор, окно проекта и т.д.)?
- Есть ли в OBS WebSocket API методы для включения вывода на проектор или это делается через настройки сцены/профиля?
