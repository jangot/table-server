# Задача: Автозапуск и процесс-менеджер

## Суть задачи

Подготовить документацию и примеры конфигов под PM2 (Linux): запуск при старте сессии, рестарт при падении, сохранение логов. При необходимости — минимальный скрипт или инструкция для автологина в Linux и запуска сервиса через PM2 после входа пользователя.

## Детали и требования

- Документация и примеры конфигов под **PM2** (Linux): запуск при старте сессии, рестарт при падении, сохранение логов.
- При необходимости — минимальный скрипт или инструкция для автологина в Linux и запуска сервиса через PM2 после входа пользователя.

## Контекст

- Документ плана: `docs/plan-execution.md`, Этап 8. Целевая ОС: Linux.
- Решение по процесс-менеджеру: PM2 (раздел 1.4 плана).
- Результат: после перезагрузки и входа пользователя сервис поднимается без ручного запуска и при падении перезапускается менеджером.
