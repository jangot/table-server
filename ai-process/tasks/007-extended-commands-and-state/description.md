# Задача: Расширенные команды и состояние

## Суть задачи

Добавить команды бота `/status`, `/idle` (переключение на URL idle-страницы), при необходимости `/restart chrome`, `/restart obs`, `/restart all`. Реализовать health-check (готовность системы, статус Chrome/OBS). Обеспечить сохранение и восстановление последнего URL при рестарте сервиса, если ещё не сделано на предыдущих этапах.

## Детали и требования

- **Команды:** `/status`, `/idle` (переключение на URL idle-страницы — локальный сервер с EJS), при необходимости `/restart chrome`, `/restart obs`, `/restart all`.
- **Health-check:** endpoint или ответ в боте: готовность системы, статус Chrome/OBS.
- Сохранение и восстановление последнего URL при рестарте сервиса (если ещё не сделано на Этапе 2/3).
